Tree
==================
This module includes various methods of building decision trees. If you have a categorical dependent variable, please use those methods that contain Classifier part in their names. 
Otherwise, if you have a scale dependent variable, please use the methods that contain Regressor part in their names.

Currently, it includes the following methods:

* Chi-square Automatic Interaction Detection a.k.a ChAID (classes CHAIDClassifier and CHAIDRegressor)
